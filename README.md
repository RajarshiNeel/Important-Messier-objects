# IOAA / TST Messier Priority Guide

An interactive web application designed for students preparing for the International Olympiad on Astronomy and Astrophysics (IOAA) and national Team Selection Tests (TST). This guide prioritizes the 110 Messier objects based on historical exam papers, observational likelihood, distinctiveness, and frequent theoretical appearances.

## About the Project

In competitive astronomy Olympiads like the IOAA, students are tested on deep-sky object (DSO) identification and observation skills. However, past papers show that not all 110 Messier objects are tested with equal frequency. 

Based on resources like "Starmaps 101" and historical competitive exams, this tool categorizes and filters Messier objects into structured priority tiers. This helps students optimize their telescope practice and theoretical preparation by focusing on high-probability targets.

---

## Key Features

- **Syllabus Overview Dashboard:** Tracks object distribution counters across Galaxies, Star Clusters, Nebulae, and Supernova Remnants.
- **Priority Filtering System:** Toggle between different learning tiers:
  - High Priority: Absolute must-knows (iconic or naked-eye objects).
  - Medium Priority: Common telescope targets frequently featured.
  - Low / Advanced Priority: Rarely tested in isolation but crucial for complete mastery.
- **Interactive Target Database:** Live search functionality to look up any Messier object instantly.
- **Detailed Object Modals:** Click on objects to reveal competition data including Apparent Magnitude, Constellation, Object Type, and Best Observing Season.
- **Preparation Insights:** Includes contextual reasoning for why an object is important for TST/IOAA along with practical star-hopping tips.

---

## Priority Tier Breakdown

The catalog distribution emphasizes structural telescope practice by grouping objects into:
- **Galaxies:** Dominating the catalog; essential for theoretical core-collapse and structure questions.
- **Star Clusters:** Open and Globular clusters frequently targeted for naked-eye and binocular simulation tests.
- **Nebulae & Remnants:** Targets for emission, reflection, and planetary nebulae identification.
